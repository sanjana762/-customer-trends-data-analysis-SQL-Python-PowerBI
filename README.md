# -customer-trends-data-analysis-SQL-Python-PowerBI
This project analyzes customer behavior data to identify trends, patterns, and insights that help in making data-driven business decisions. SQL is used for data querying, Python is used for data cleaning and analysis, and Power BI is used to design an interactive dashboard for visualization.

The goal is to understand customer purchasing habits, preferences, and segmentation to improve marketing strategies and customer engagement.
